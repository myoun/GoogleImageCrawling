# Google Image Crawling Program

## 사용방법

### * Windows
`google.py -s python -f python_img`

### * Linux / Mac
`python google.py -s python -f python_img`

### Arguments

`-s` 나 `--search` 뒤에는 검색어를 붙여주세요 (필수)<br/>
`-f` 나 `--filename` 뒤에는 파일 이름을 붙여주세요 [ ex) 파이썬 -> 파이썬1.jpg ] ( 선택, 안붙이면 검색어가 파일 이름이 됩니다. )
